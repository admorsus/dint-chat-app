el archivo whatsapp.sql  sera la base de datos exporta desde phpmyadmi.

hemos consegido desarrollar los sigietes metodos: 

connection: este metodo comprueba que exista una conexion a la base de datos

login_ok: esta funcion compruba que el usuario que va a hacer login este registrado en la base de datos, si no lo esta es imposible hacer login.

guardar_mensaje: esta funcion guarda en la base de datos los mensajes que envia el usuario a otro usuario.

perfil_modificado: modifica los datos del perfil de la base de datos de la tabla usuario.
